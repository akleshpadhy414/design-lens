## Mandatory Copy Rules to Enforce

### Capitalization
- ALL UI text must be sentence case. No Title Case anywhere except proper nouns.
- Proper nouns that ARE capitalized: HighLevel, Google, Facebook, Funnel AI
- Headings: sentence case ("Create new campaign" NOT "Create New Campaign")
- Buttons: sentence case ("Save changes" NOT "Save Changes")
- Column headers: sentence case
- Tooltips: sentence case
- Error/success messages: sentence case
- Flag EVERY instance of incorrect capitalization as an issue.

### Buttons & Actions
- Buttons must be verb-first, sentence case ("Save changes", "Add contact", "Export data")
- FORBIDDEN button labels — flag these as severity "error" with suggested replacement:
  - "Submit" → use "Save changes", "Send", "Create", or a specific action verb
  - "OK" → use "Confirm" or "Continue"
  - "Click Here" → use "Learn more" or a specific action
  - "Retry" → use "Try again"
  - "Cancel" in billing context → use "Go back" (conflicts with subscription cancellation)
- Generic labels like "Continue" should be replaced with specific action descriptions

### Errors & Alerts
- Must be helpful and calm, not technical or alarmist
- NEVER show technical errors to users. Flag any instance of:
  - Error codes ("Error 503", "Error 404", "HTTP 500")
  - Technical messages ("Bad Gateway", "Null reference", "Timeout")
- DO suggest user-friendly alternatives: "Can't connect right now. Check your internet and try again."
- Error messages must focus on what the user CAN DO next

### Filler Phrases — Flag and Remove
- Flag ANY instance of these filler phrases as severity "warning":
  - "Please note that..."
  - "Kindly..."
  - "Click here to begin..."
  - "In order to..."
  - "It should be noted that..."
  - Any unnecessarily wordy phrasing
- Suggest concise replacements

### Number Formatting
- Numbers must use numerals: "3 campaigns" not "three campaigns"
- Thousands: use commas (1,000; 12,500)
- Large numbers: abbreviate when space-limited (1.2K, 3.5M)
- Percentages: numerals with % (5%, 12.5%)

### Currency
- Use $ with numerals: $49, $1,299
- Use .00 only when precision matters

### Dates
- US format: MM/DD/YYYY (01/14/2026)
- Readable format for headings: Jan 14, 2026
- Time: 12-hour with space before AM/PM (3:00 PM)
- Time ranges: en dash (3:00–4:00 PM)

### Phone Numbers
- Format: (###) ###-#### — example: (415) 555-0132
- With country code: +1 (###) ###-####
- NO dots ever (415.555.0132 is WRONG — flag as error)

### Punctuation
- Oxford comma required (A, B, and C)
- Periods: use in full sentences, omit in short labels and buttons
- Exclamation marks: only for positive reinforcement ("All changes saved!")
- Ellipses: only for incomplete actions ("Loading...")

### Empty States
- Must be concise and actionable with a clear next step
- Pattern: brief description + action button
- Example: "You don't have any templates yet. Create your first one to get started."

### Tone
- Clear, confident, friendly — no jargon
- Short, actionable, human
- No filler, no corporate speak
